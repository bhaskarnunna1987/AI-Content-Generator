# AI Content Generator

## Project Overview

This project is an AI Content Generator developed as part of the WeIntern AIML Week 2 Internship Assignment.

The application generates blog posts and social media captions based on user-provided topics and selected tones.

## Features

* Generate Blog Posts
* Generate Social Media Captions
* Topic Input
* Tone Selection
* Content Type Selection
* Word Count Display
* Character Count Display
* Error Handling for Empty Inputs

## Technologies Used

* Python
* Streamlit

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the application:

python -m streamlit run app.py

3. Open the Streamlit URL in your browser.
